# Portfolio

A modern, 3D animated portfolio website built with React, Vite, Tailwind CSS, Framer Motion, and Three.js.

## Features

- **3D Background**: Immersive star field using `@react-three/drei`.
- **Hero Section**: Animated text and 3D distorting sphere.
- **Skills**: 3D rotating cards with hover effects.
- **Projects**: Interactive project cards with tilt effects.
- **Experience**: Timeline view of internships and education.
- **Contact**: Functional contact form (UI only) and social links.
- **Smooth Scroll**: Powered by `lenis` for a premium feel.
- **Responsive**: Fully responsive design for all devices.

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

## Customization

-   Update `src/constants/index.js` with your own data.
-   Modify `tailwind.config.js` to change the color theme.
-   Adjust animations in individual components using Framer Motion props.

## Tech Stack

-   React 18
-   Vite
-   Tailwind CSS
-   Framer Motion
-   Three.js / React Three Fiber / Drei
-   Lenis (Smooth Scroll)
